Utilização do sistema

Este texto é um EXEMPLO de material de ajuda para a versão de desenvolvimento do sistema LogicITS.

Esta aplicação apresenta exercícios de Dedução Natural na Lógica Proposicional. Para utilizá-la, o estudante tem a disposição um conjunto de botões referentes às Regras de Inferência.

Uma resolução é chamada de prova. Ao selecionar uma das regras de inferência, é necessário especificar em qual(is) linha(s) da prova deseja aplicar a regra. Para isso, basta selecionar uma ou mais - a quantidade depende da regra selecionada - das linhas da lista mostrada em tela.

Além das regras de inferência, é possível iniciar uma hipótese. Para isso, deve-se selecionar o botão "Adicionar hipótese" e inserir na caixa de texto a fórmula desejada.

Caso o estudante viole alguma das restrições de utilização das regras de inferência, será notificado através do painel de Feedback. A sequência de atividades é gerada automaticamente de acordo com o desempenho do estudante.
